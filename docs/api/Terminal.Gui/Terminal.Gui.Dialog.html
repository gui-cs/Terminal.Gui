    Class Dialog

    The Dialog View is a Window that by default is centered and contains one or more Buttons. It defaults to the Dialog color scheme and has a 1 cell padding around the edges.

    Remarks
    To run the Dialog modally, create the Dialog, and pass it to Run(Func<Exception, Boolean>). This will execute the dialog until it terminates via the [ESC] or [CTRL-Q] key, or when one of the views or buttons added to the dialog calls RequestStop().

    Remarks

    Te Dialog will be vertically and horizontally centered in the container and the size will be 85% of the container. After initialzation use X, Y, Width, and Height to override this with a location or size.

    Use AddButton(Button) to add buttons to the dialog.
