    HotKey

    The HotKey is used when the menu is active, the shortcut can be triggered when the menu is not active. For example HotKey would be "N" when the File Menu is open (assuming there is a "_New" entry if the Shortcut is set to "Control-N", this would be a global hotkey that would trigger as well
