# Plan: Create Compressed API Documentation Index

## Goal

Create a Vercel-style compressed documentation index for AGENTS.md that includes **all API reference content** from the C# XML documentation (not just deep dives).

## Background

- Vercel's approach: Create a retrieval-based INDEX pointing to documentation files, not a summary
- Current state: We have an index for deep dives (docfx/docs/) but missing the full API reference
- The API reference is generated by docfx from XML documentation comments in .cs files
- Live docs: https://gui-cs.github.io/Terminal.Gui/api/

## Steps

### 1. Build the Documentation Site

```bash
cd /path/to/Terminal.Gui

# Restore and build
dotnet restore
dotnet build Terminal.Gui/Terminal.Gui.csproj --configuration Release

# Install docfx and generate docs
cd docfx
dotnet tool install -g docfx
docfx metadata  # Generates ./api/*.yml from DLL
docfx build     # Generates ./_site with full HTML docs
```

### 2. Extract API Types from Generated Metadata

After docfx runs, examine:
- `docfx/api/*.yml` - YAML metadata for all types
- `docfx/_site/api/*.html` - Generated HTML pages

Key namespaces to index:
- `Terminal.Gui.App` - Application, IApplication, IRunnable, SessionStack, Clipboard, etc.
- `Terminal.Gui.ViewBase` - View, Pos, Dim, Adornment, Border, Margin, Padding, etc.
- `Terminal.Gui.Views` - Button, Label, TextField, ListView, CheckBox, Dialog, Window, etc.
- `Terminal.Gui.Input` - Key, Mouse, Command, KeyBindings, MouseBindings, etc.
- `Terminal.Gui.Drawing` - Attribute, Color, LineCanvas, Cell, Glyphs, etc.
- `Terminal.Gui.Drivers` - IDriver, DriverRegistry, platform drivers
- `Terminal.Gui.Configuration` - ConfigurationManager, themes, settings
- `Terminal.Gui.Text` - Text processing, autocomplete
- `Terminal.Gui.FileServices` - File dialogs

### 3. Create Compressed Index Format

Follow Vercel's pipe-delimited format:

```
[Terminal.Gui.App]|root:docfx/_site/api
|Application|Static entry point,Init,Run,Shutdown
|IApplication|Instance-based app,TopRunnable,SessionStack,Dispose
|IRunnable<T>|Typed result views,Result property
|SessionToken|Running session handle
|Clipboard|OS clipboard support
|TimedEvents|Scheduled callbacks
|CWPPropertyHelper|Property change CWP
|CWPWorkflowHelper|Workflow CWP
...

[Terminal.Gui.ViewBase]
|View|Base class,Frame,Viewport,ContentArea,Add,Remove
|Pos|Position:Absolute,Percent,Center,AnchorEnd,Align,Combine
|Dim|Dimension:Absolute,Fill,Percent,Auto,Combine
|Adornment|Outside viewport drawing
|Border|Visual frame,Title
|Margin|Outer spacing
|Padding|Inner spacing
...
```

### 4. Update Files

1. **`.tg-docs/INDEX.md`** - Detailed index with all types
2. **`AGENTS.md`** - Add compressed API reference section

### 5. Commit and Push

```bash
git add .tg-docs/ AGENTS.md
git commit -m "Add comprehensive API reference to compressed docs index"
git push -u origin claude/compress-api-docs-v2-A62iW
```
